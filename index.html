Quick Start

Requires llama.cpp built with a compatible backend and a GGUF model.

./run.sh

Or invoke directly:

./bin/llama-cli \
  -cnv \
  -t 4 \
  --ctx-size 8192 \
  --temp 0 \
  --color on \
  -m /path/to/model.gguf \
  -p "$(cat prompt.txt)"

Runtime Parameters

ParameterValueRationale

-t4Four threads map to the four Anchor Spark quadrants
