## Quick Start

### Prerequisites

- [llama.cpp](https://github.com/ggerganov/llama.cpp) built with a compatible backend
- A GGUF-format model (e.g. `gemma-3-1b-it-Q4_0.gguf`)

### Run

```bash
./run.sh
```

Or invoke directly:

```bash
./bin/llama-cli \
  -cnv \
  -t 4 \
  --ctx-size 8192 \
  --temp 0 \
  --color on \
  -m /path/to/model.gguf \
  -p "$(cat prompt.txt)"
```

See [`run.sh`](run.sh) and [`prompt.txt`](prompt.txt) for the pre-configured parameters.
